Install Instructions:

Our current version requires ANSITerminal

Please update OPAM before installing:
command: opam update

This first command may take a while, but that is normal.
Next, you may have to upgrade already-installed packages
You can do so with the following command:
command: opam upgrade

Finally, you can install ANSITerminal.
command: opam install ANSITerminal

That should be all the installations you will need!


To play the game:

Make sure your directory is inside the chess folder.
You can do so by running the following command:
command: cd chess

To run our game and play the UI, do
command: make play
